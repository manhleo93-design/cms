# CMS - Quản lý liên kết nội bộ

Đây là một hệ thống quản lý liên kết nội bộ (Backlink Manager) tối giản được phát triển với các công nghệ:
- PHP 8+
- MySQL
- Tailwind CSS
- AJAX (Fetch API).

Dự án được xây dựng với mục tiêu làm rõ các tính năng cơ bản, gồm CRUD campaign và backlink, kiểm tra trạng thái AJAX không tải lại trang. Ngoài ra giao diện light theme gọn gàng, hiện đại phong cách dashboard SaaS là điểm nhấn.

## Cách cài đặt
1. Tạo cơ sở dữ liệu MySQL.
2. Chạy tệp `install.php` để khởi tạo cơ sở dữ liệu.
3. Truy cập `index.php` từ trình duyệt để sử dụng.
